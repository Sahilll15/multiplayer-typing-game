# ⌨️ Typing Titans — Multiplayer Typing Game

**Race players around the world in real time** to type a passage fastest. An adrenaline-pumping multiplayer typing showdown.

## ✨ Features

- 🌐 **Real-time multiplayer** rooms powered by **Socket.IO**
- 🏁 **Live race** — see every player's progress & WPM as they type
- 🏆 Winner detection and results
- 📱 Responsive UI

## 🗂️ Project Structure

```
multiplayer-typing-game/
├── client/   # React frontend
└── server/   # Node.js + Express + Socket.IO (TypeScript) + MongoDB
```

## 🚀 Getting Started

```bash
git clone https://github.com/Sahilll15/multiplayer-typing-game.git
cd multiplayer-typing-game

# --- Server ---
cd server
npm install
# .env → MONGO_URI, PORT
npm run dev

# --- Client (new terminal) ---
cd ../client
npm install
npm start
```
